# Recipe Generator

A fun app that suggests recipes based on ingredients you have!

## Live Demo
https://recipe-generator-1-dqzo.onrender.com

## Features
- Add and remove ingredients
- Generate creative recipes using AI (powered by Qwen2.5)
- Clean, modern interface

## How to run locally
1. Clone the repo
2. `npm install`
3. `npm run dev` (frontend)
4. In another terminal: `node server.js` (backend)
